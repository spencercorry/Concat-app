How to use DlimitR

Input

Paste your list into the Input List box — one item per line. Duplicates are removed and whitespace is trimmed automatically.

Delimiter

SQLString — wraps each item in single quotes: 'item'

SemiColon — appends a semicolon: item;

Comma — separates items with commas: item,

Wrapper

Surrounds the entire output in ( ) or { }. Useful for SQL IN (...) clauses.

Wrap Value

Sets how many items appear on each line before a line break is inserted.

Output

Click Format List to run. Click the output area to select all text, or hit Copy to copy it to your clipboard.
